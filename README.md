# Mp3tag tooling

```
NAME:
   mp3tag - Mp3Tag

USAGE:
   mp3tag [global options] command [command options] [arguments...]

VERSION:
   1.0

COMMANDS:
     fileNameToTitle           Set file name to title
     fileNamePrefixToTitle     Set file name prefix, separated by separator, to title
     fileNamePrefixToFileName  Set file name prefix, separated by separator, to file name
     help, h                   Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --folder value, -f value  Path of folder with mp3 files (default: ".")
   --help, -h                show help
   --version, -v             print the version
```
